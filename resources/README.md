# Resources

Repository for workshop materials and source references.

## Structure

- `links.md`: curated links by topic.
- `registry.csv`: machine-readable index of resources.
- `papers/`: open-access PDFs only.

## Public-safety rules

- Do not commit private notes with sensitive data.
- Do not upload copyrighted PDFs without redistribution rights.
- Prefer arXiv, working paper, and open-access publisher versions.
