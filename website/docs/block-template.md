# Block Template

Use this for every 60-minute teaching block.

## Structure

- Objective
- Inputs
- Deliverable (single micro artifact)
- Timebox (20/30/10)
- Submission link (prefilled `exercise_id`)
- Evaluation rubric
- Common failure modes
- Extension task (optional)

## Timebox

- 20 minutes lecture input
- 30 minutes hands-on activity
- 10 minutes debrief and critique

## Debrief prompts

- What assumption failed?
- What evidence would falsify this?
- Where could this hallucinate?
- What should not be delegated?
- What is the cheapest simplification?
