## Teaching controls

- Add `?exercise=E3` to highlight one active exercise.
- Add `&start=2026-03-09T09:30:00` to show a countdown for the active card.
- Add `&group=G4` to prefill a group code.

## Setup

Set these variables before build/deploy:

```text
GOOGLE_FORM_URL=https://docs.google.com/forms/d/e/<FORM_ID>/viewform?usp=pp_url
GOOGLE_FORM_EXERCISE_FIELD=<EXERCISE_FIELD>
GOOGLE_FORM_GROUP_FIELD=<GROUP_FIELD>   # optional
ACTIVE_EXERCISE_ID=E1                   # optional
```

## Fallback

- Share the direct form link.
- Use a QR code from the same URL.
- If needed, collect one spokesperson response per group.

## Required telemetry fields

To support live debrief diagnostics, ask groups to always submit:

- confidence (1-5)
- one uncertainty/failure note
- one verification method used
