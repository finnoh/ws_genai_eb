# Self-Study Project Studio

<div className="courseHero">
  <p className="courseKicker">Project Paths</p>
  <p className="courseLead">Choose one project type and ship a reproducible artifact with explicit reliability checks.</p>
</div>

<div className="courseGrid">
  <section className="courseCard">
    <h3>Path A: Prompt evaluation pack</h3>
    <p>Build a small benchmark of prompts for one research task.</p>
    <ul>
      <li>Inputs: 10 prompts, one scoring rubric</li>
      <li>Output: ranked prompt set with evidence</li>
      <li>Risk check: disagreement and drift notes</li>
    </ul>
  </section>
  <section className="courseCard">
    <h3>Path B: Coding-agent reliability report</h3>
    <p>Run repeated coding-agent sessions on the same issue type.</p>
    <ul>
      <li>Inputs: issue list and acceptance tests</li>
      <li>Output: trace-based quality report</li>
      <li>Risk check: rollback and verification failures</li>
    </ul>
  </section>
</div>

<div className="courseGrid">
  <section className="courseCard">
    <h3>Path C: Agent workflow blueprint</h3>
    <p>Design a tool-and-memory contract for one end-to-end pipeline.</p>
    <ul>
      <li>Inputs: task boundary and constraints</li>
      <li>Output: architecture plus fallback matrix</li>
      <li>Risk check: human handoff criteria</li>
    </ul>
  </section>
  <section className="courseCard">
    <h3>Path D: Adoption decision memo</h3>
    <p>Evaluate two candidate workflows and choose one deployment strategy.</p>
    <ul>
      <li>Inputs: cost, quality, and risk data</li>
      <li>Output: recommendation memo</li>
      <li>Risk check: governance and monitoring plan</li>
    </ul>
  </section>
</div>
