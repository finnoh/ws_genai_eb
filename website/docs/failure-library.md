# Failure Library

Use this during debrief to diagnose recurring agent mistakes.

## Common failures

- Ambiguous instructions
- Missing constraints
- False confidence
- Poor tool selection
- Unverifiable claims
- Over-engineered multi-agent designs

## Debrief prompts

- What assumption failed?
- What evidence would falsify this?
- Where could this hallucinate?
- What should not be delegated?
- What is the cheapest simplification?

## Quick fix checklist

- Tighten task boundary
- Add explicit output schema
- Add verification step
- Add fallback and escalation path
