# Self-Study Track

You can complete this material without attending the live workshop.

Use this path if you want both learning outcomes and portfolio-ready artifacts.

## Suggested path (3-5 hours)

1. Read [Overview](/docs/overview) and [Schedule](/docs/schedule).
2. Watch Day 1 and Day 2 slides in [Slides and Assets](/docs/slides-and-assets).
3. Complete exercises E1, E2, E5, and E7 from [Live Exercises](/docs/live-exercises).
4. Compare your outputs using [Live Results](/docs/live-results) format.
5. Reuse templates from [Patterns](/docs/patterns) and [Failure Library](/docs/failure-library).
6. Build your own mini project using [Resources](/docs/resources).

## Portfolio outputs to produce

- One prompt rewrite with failure analysis
- One coding-agent trace with verification
- One agent workflow diagram and rationale
- One evaluation memo comparing two outputs

## Optional extension

- Adapt the workflow to your own research topic
- Add your own rubric and reliability checks
