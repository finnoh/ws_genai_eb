# Block Drafts

These are the working drafts for each scheduled teaching block, aligned to `AGENTS.md`.

If a deck says "not found", render slides first from repo root:

```bash
./scripts/render_slides.sh
```

Shared defaults for every block:

- Timebox: 20/30/10
- Evaluation: usefulness, correctness, reproducibility, risk awareness
- Debrief prompts: failed assumption, falsification evidence, hallucination risk, non-delegable step

## Day 1 - Foundations

- Deck: <DeckLink path="/slides/blocks/day1/day1_b1_orientation.html" label="D1 B1 deck" />
