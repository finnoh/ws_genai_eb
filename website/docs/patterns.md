# Patterns

Reusable templates to reduce prompt-specific guesswork.

## Prompt decomposition template

- Task goal
- Inputs and constraints
- Step-by-step plan
- Output format
- Verification check

## Tool contract template

- Tool name and purpose
- Required inputs
- Expected outputs
- Failure handling
- Human override condition

## Evidence-check template

- Claim
- Supporting source
- Confidence level
- What would falsify claim

## Agent handoff template

- Upstream artifact
- Handoff instruction
- Acceptance criteria
- Escalation trigger
