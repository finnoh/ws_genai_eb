# Live Hub

Keep this page open during class. It is the quickest route from task launch to submission to debrief.

## Classroom flow

1. Confirm the active block in [Schedule](/docs/schedule).
2. Open the active card in [Live Exercises](/docs/live-exercises).
3. Submit one group artifact through the form.
4. Review examples on [Live Results](/docs/live-results).

## If you are joining now

- Complete setup checks in [Start Here](/docs/start-here).
- Use one shared group code for your table.
- Keep one spokesperson responsible for final submission.

## Facilitator shortcuts

- `?exercise=E3` highlights one active exercise card.
- `&start=2026-03-09T09:30:00` starts a visible countdown.
- `&group=G4` prefills a group code for quick launches.

## Fallback

- Share the direct Google Form link if cards fail.
- Use one spokesperson submission per group if time is tight.
- Open the response sheet directly if live table refresh fails.
