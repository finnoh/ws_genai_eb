# Workshop Overview

This companion website supports the 2-day Tinbergen Institute workshop:

**AI Agents in Economics and Business Research**

## Learning goals

- Use LLM agents in research workflows.
- Build reliable, testable agent routines.
- Apply safety, privacy, and sustainability checks.

## Structure

- **Day 1**: foundations and safe practice (basic -> intermediate)
- **Day 2**: advanced systems and research integration (intermediate -> advanced)
- **Every block**: 20 min lecture, 30 min hands-on, 10 min debrief

## Storyline

- Day 1 builds conceptual and workflow basics.
- Day 2 applies those basics to robust multi-step research workflows.
- Each block ends with one micro artifact used in later blocks.

## Classroom workflow

- Open **Live Hub** and follow the active block.
- Launch each exercise from **Live Exercises**.
- Submit group output through the workshop form.
- Review submissions together in **Live Results**.

## Teaching operating system

- Observe -> Build -> Fail -> Verify -> Reflect
- One micro artifact per block
- Adversarial debrief prompts over presentation summaries
