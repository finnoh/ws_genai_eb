Exercise Brief

E01 - Getting started: Setting up Jan, API keys, and LangChain Hello World!

In this first exercise, you get your local agent environment working end-to-end. The goal is not advanced coding: it is operational confidence. By the end, you will have Jan running, one tiny LangChain task completed, and one manual verification so you know what “working” looks like.

  • Deliverable: Minimum: one screenshot/terminal log of successful startup plus one Jan response. Full: add one CSV hello-world run with manual verification and a short note (2-3 sentences) on one Jan customization you enabled and why.
  • Timebox: 20/30/10
  • Submit: E01
