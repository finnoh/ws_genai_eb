# Exercise Submission Draft

## Objective
Diagnose a weak prompt and produce an improved version.

## Key decision
Force the model to produce a verifiable, source-grounded output format instead of an unconstrained narrative answer.

## Likely failure mode
The rewritten prompt may still be too broad, causing generic claims without evidence.

## Mitigation
Add a strict output schema with claim-to-evidence mapping and a confidence tag for each claim.

## Verification method
Run a two-pass check: (1) verify each claim has a cited source snippet and (2) remove any claim that cannot be traced to provided sources.

## Final response

Weak prompt (before):
"Explain whether AI agents improve research productivity."

Improved prompt (after):
"You are a research assistant for economics and business methods. Using only the provided sources, evaluate whether AI agents improve research productivity.

Task constraints:
1) Return exactly 3 claims.
2) For each claim, include: claim, evidence quote, source ID, confidence (high/medium/low), and one limitation.
3) If evidence is missing, write 'insufficient evidence' instead of guessing.
4) End with one falsifiable test that could disconfirm your conclusion.

Output format:
- Claim 1: ...
  - Evidence: ...
  - Source: ...
  - Confidence: ...
  - Limitation: ...
- Claim 2: ...
- Claim 3: ...
- Falsification test: ..."

Rationale:
The rewrite improves usefulness (clear target), correctness (evidence-bound), reproducibility (fixed structure), and risk awareness (limitations plus falsification).
