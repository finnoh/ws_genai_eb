# Exercise Rubrics

Scale for all criteria: 0 (missing), 1 (weak), 2 (acceptable), 3 (strong).

## Canonical base rubric (use for every exercise)

- Usefulness: does the output help the stated task?
- Correctness: is it factually/procedurally sound?
- Reproducibility: can another group rerun it?
- Risk awareness: are key risks identified and mitigated?

- Correct execution evidence
- Quality of Jan customization note
